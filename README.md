## Requirements

Open `mnist.ipynb` in a Jupyter Notebook environment.

Using Python 3.13.1, ensure the following packages are installed:

TensorFlow 2.20.0
```
pip install tensorflow
```
Keras 3.13.2
```
pip install keras
```
numpy 2.4.2
```
pip install numpy
```
matplotlib 3.10.8
```
pip install matplotlib
```

Then run all code cells. Outputs will be displayed under their respective code cells.
